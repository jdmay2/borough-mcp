# Security Policy

If you believe you have found a security issue in Borough MCP or its public package surface, email [api@qwady.com](mailto:api@qwady.com).

## Scope

- This public repository covers the MCP bridge package and public connector docs.
- The live Borough service runs separately on Cloudflare Workers and is not deployed from this repository.
- All MCP tools exposed by Borough are read-only.

## Reporting

Please include:

- a concise description of the issue
- steps to reproduce
- expected vs actual behavior
- any affected versions or environments

Please do not publish exploit details before we have had a reasonable chance to investigate.
